# Path to Success 🏍️

An interactive celebration for the Sword Health Growth team. Ride a motorcycle
up three years of real weekly enrollment data (Jul '23 → Jul '26, 184 → 12,904/week).
Every all-time high you cross explodes in celebration; the end of the road says thank you.

**Controls:** hold `Space` / click / touch to accelerate. `M` mute, `R` restart.

Turn the sound on — the synth-disco soundtrack (a Giorgio Moroder homage, synthesized
live in WebAudio) adds a layer every time a record falls.

Everything is one self-contained `index.html` — no dependencies, no build step.

## Run locally

Open `index.html` in any browser, or:

```sh
python3 -m http.server 8000   # then visit http://localhost:8000
```

## Deploy to Vercel

```sh
npx vercel --prod
```

…or drag this folder onto [vercel.com/new](https://vercel.com/new). It's a static
site — zero configuration needed.
